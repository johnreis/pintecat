# pintecat
Pintecat, the app that fetches images of cats from http://api.imgur.com

### Install dependencies with [CocoaPods](https://cocoapods.org/#install)

``` shell
$ pod install
```

### Pods
- 'Alamofire', '~> 5.2'
- 'AlamofireImage', '~> 4.1'
- 'RxSwift', '~> 5'
- 'RxCocoa', '~> 5'
- 'RxTest', '~> 5'
- 'RxBlocking', '~> 5'
- 'SkeletonView'
- 'Cuckoo'
